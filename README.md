# Miner's Escape

Miner's Escape is a 3D cave adventure game where the player explores underground areas, gathers minerals, fights goblins, and tries to survive while progressing deeper into the mine.

## About the Game

The main goal of the game is to collect minerals, defend yourself from goblins, and continue through the cave by unlocking progression with the resources you gather. The gameplay combines exploration, combat, and resource collection.

The player can switch between two tools:
- a **pickaxe** for mining minerals
- a **pistol** for fighting goblins

To succeed, the player must balance collecting resources with surviving enemy attacks.

## How to Play

- Explore the cave and search for minerals.
- Use the **pickaxe** to mine and gather materials.
- Use the **pistol** to defend yourself against goblins.
- Collect enough minerals to continue progressing.
- Stay alive and complete the final objective to win the game.

## Controls

- **W / A / S / D** – movement
- **Mouse** – camera / aiming
- **Left Shift** – run
- **Left Mouse Button** – dig or shoot
- **1** – equip pickaxe
- **2** – equip pistol
